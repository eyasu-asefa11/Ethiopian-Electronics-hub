## 🚀 **STEP-BY-STEP SEQUENCE**

### **📱 STEP 1: Open Terminal/Command Prompt**
```
📱 Windows: Open Command Prompt or PowerShell
🔧 Mac: Open Terminal
📱 Linux: Open Terminal
🔧 Navigate to your project folder
```

### **📱 STEP 2: Navigate to Project Directory**
```bash
# Navigate to your project folder
cd "c:\Users\eyasu\OneDrive\Documents\dilla-electronics-marketplace"

# Verify you're in the right directory
dir  # On Windows
ls   # On Mac/Linux
```

### **📱 STEP 3: Install Backend Dependencies**
```bash
# Navigate to backend folder
cd backend

# Install backend packages
npm install

# Wait for installation to complete
# This will install: express, sqlite3, cors, bcrypt, jsonwebtoken, multer, better-sqlite3
```

### **📱 STEP 4: Start Backend Server**
```bash
# Start the backend server
npm start

# OR if the above doesn't work, try:
node server.js

# You should see:
# 🚀 Server running on port 5000
# 📱 Ethiopian Electronics Marketplace Backend Started
```

### **📱 STEP 5: Open NEW Terminal Window**
```
📱 Keep the backend terminal OPEN
🔧 Open a NEW terminal window (don't close the first one)
📱 This new terminal will be for the frontend
🔧 Navigate to the project directory again
```

### **📱 STEP 6: Navigate to Frontend Directory**
```bash
# In the NEW terminal, navigate to frontend folder
cd "c:\Users\eyasu\OneDrive\Documents\dilla-electronics-marketplace\frontend"

# Verify you're in the right directory
dir  # On Windows
ls   # On Mac/Linux
```

### **📱 STEP 7: Install Frontend Dependencies**
```bash
# Install frontend packages
npm install

# Wait for installation to complete
# This will install: React, react-router-dom, axios, testing libraries, etc.
```

### **📱 STEP 8: Start Frontend Development Server**
```bash
# Start the frontend development server
npm start

# This will automatically open your browser
# You should see:
# 📱 Starting the development server...
# 🚀 Compiled successfully!
# 📱 Local: http://localhost:3000
```

## 🛠️ **TROUBLESHOOTING**

### **❌ Common Issues & Solutions:**

#### **📱 Issue: "Port 3000 is already in use"**
```bash
# Solution 1: Kill the process
npx kill-port 3000

# Solution 2: Use different port
npm start -- --port=3001
```

#### **📱 Issue: "npm command not found"**
```bash
# Solution: Install Node.js from https://nodejs.org
# Restart your terminal after installation
```

#### **📱 Issue: "Module not found"**
```bash
# Solution: Make sure you're in the right directory
cd backend    # For backend
cd frontend   # For frontend

# Then run npm install again
npm install
```

#### **📱 Issue: "Cannot read property of undefined"**
```bash
# Solution: Clear npm cache and reinstall
npm cache clean --force
npm install
```

#### **📱 Issue: Backend not connecting**
```bash
# Solution: Check if backend is running
# Open new tab in browser and go to: http://localhost:5000
# If you see "Cannot GET /" - backend is running
# If you see "This site can't be reached" - backend is not running
```
