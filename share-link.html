📋 How to Share with Customers:

  1. Copy the Link: Click "Copy Link" button above
  2. Share via WhatsApp: Click "Share via WhatsApp" to send directly
  3. QR Code: Customers can scan the QR code with their phone
  4. Local Network: Make sure you're both on the same WiFi/network
  5. Keep Server Running: Your development server must be running
⚠️ Important Note: This link works only when your development server is running. For production deployment, you'll need to host this online.

🌐 Network Setup for External Access:

  1. Find Your IP Address: Open Command Prompt and run: ipconfig
  2. Use Your IP: Replace "localhost" with your IP address
  3. Example: http://192.168.1.100:3000
  4. Firewall: Make sure port 3000 is allowed
